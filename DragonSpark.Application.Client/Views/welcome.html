Welcome

Welcome to The DragonSpark Featureset Application, hosted on Microsoft's Azure Servers.

Features

Here are a few of the key development friction points that The DragonSpark Application Framework addresses:
  • Application Initialization
  • Xaml-based Application Configuration (IoC and WebApi Controller Configuration)
  • Operation Management and User Experience
  • Exception Handling, Reporting, and User Experience
  • Authentication and Claims Transformation (from external identity providers)
  • Authorization
  • Navigation
  • User Profile Initialization and Management
  • Xaml-based Entity Installation (Versioned Database seeding -- no more Sql Scripts)
  • Entity Management

Technologies

We are proud to make use of the following client-side technologies:
  • Durandal
  • Knockout 3.0
  • SignalR 2.0
  • BreezeJS
And the following server-side technologies:
  • eXtensible Application Markup Language (XAML) (On the server? Yes. On the server. Here's an example.)
  • ASP.NET MVC 5.0
  • Entity Framework 6.0 Code First
  • Microsoft Patterns and Practices' Unity for Service Location and IoC
  • Microsoft Patterns and Practices' Enterprise Library 6.0
  • Enterprise Library Semantic Logging Application Block
