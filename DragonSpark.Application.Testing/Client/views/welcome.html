Welcome

Welcome to The DragonSpark Featureset Application, hosted on Microsoft's Azure Servers.

Features

Here are a few of the key friction points that The DragonSpark Application Framework addresses:
  • Application Initialization
  • Application Configuration (Xaml-Driven IoC)
  • Operation Management and User Experience
  • Exception Handling, Reporting and User Experience
  • Authentication and Claims Transformation (from external identity providers)
  • Authorization
  • Navigation
  • User Profile Initialization and Management
  • Entity Installation (Versioned Xaml-Driven Database seeding -- no more Sql Scripts)
  • Entity Management

Technologies

We are proud to make use of the following technologies:
  • Durandal
  • jQuery Widgets
  • Entity Framework CodeFirst
  • Microsoft Patterns and Practices' Unity for Service Location and IoC
  • Microsoft Patterns and Practices' Enterprise Library
  • Windows Azure
  • Windows Azure Access Control 2.0
